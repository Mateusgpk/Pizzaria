# Sistema de Pedidos — Pizzaria 

Este projeto é um sistema de gerenciamento de pedidos para uma pizzaria inovadora que comercializa pizzas por centímetro quadrado, com formatos geométricos de Círculo, Quadrado ou Triângulo. O preço de cada pizza é calculado dinamicamente com base na área total e na categoria dos sabores escolhidos.

O software foi desenvolvido em **Java** com interface gráfica **Java Swing**, aplicando conceitos de herança e polimorfismo.
